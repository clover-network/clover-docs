# web3\_sha3

```
web3.sha3(SHA3hash)
```

**Parameters**

the data to convert into a SHA3 hash

**Returns**

The SHA3 result of the given string

**Example**

```
const sha3 = web3.sha3("0x68656c6c6f20776f726c64");
console.log(sha3)
> 0x47173285a8d7341e5e972fc677286384f802f8ef42a5ec5f03bbfa254cb01fad
```
